26/* The network library requires a simple AES implementation for its
27 * IEEE 802.15.4 security. The expectation is that 802.15.4 devices will have
28 * some hardware acceleration, which should be made available through this API.
29 *
30 * If no hardware acceleration is available, the API can be easily hooked up to
31 * an external software library such as mbed TLS.
32 *
33 * To simplify porting, the API is used single-threaded, so that no special care
34 * is needed to maintain global context or state in the hardware accelerator.
35 */
