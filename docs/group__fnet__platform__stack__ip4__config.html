◆ FNET_CFG_IP4_FRAGMENTATION

#define FNET_CFG_IP4_FRAGMENTATION   (0)

IP fragmentation:

  • 1 = is enabled. The IP will attempt to reassemble the IP packet fragments and will be able to generate fragmented IP packets.
  • 0 = is disabled (Default value). The IP will silently discard the fragmented IP packets..
