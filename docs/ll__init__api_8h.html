Initialization conditional compilation are used to control LL initialization options. Define one or more of the following to enable roles and features.

  • INIT_BROADCASTER (default)
  • INIT_OBSERVER
  • INIT_PERIPHERAL
  • INIT_CENTRAL
  • INIT_ENCRYPTED
  • BT_VER
Note
Each feature may require additional LlRtCfg_t requirements.
