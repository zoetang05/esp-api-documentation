◆ ns_time_api_time_configuration_notify_callback

typedef void ns_time_api_time_configuration_notify_callback(timezone_info_t *info_ptr)

New time zone and daylight saving time information learned from stack.

Called when network stack learns the time zone and daylights saving time information.

In Wi-SUN Border router can distribute this information to synchronize all nodes in network. This information is updated roughly once a day using DHCPv6 protocol

If network stack does not have the information this is not called and can be called when information becomes available. This can be called multiple times with same information.

Parameters
info_ptrtime zone and daylight saving time configuration.
