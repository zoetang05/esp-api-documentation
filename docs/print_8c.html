#define PRINT_IS_XDIGIT ( c)
Value:
(((c >= '0') && (c <= '9')) || ((c >= 'a') && (c <= 'f')) || \
((c >= 'A') && (c <= 'F')))

Detect hexadecimal digits.

◆ PRINT_XDIGIT_TO_INT

#define PRINT_XDIGIT_TO_INT ( c)
Value:
(((c >= '0') && (c <= '9')) ? (uint8_t)(c - '0') : \
((c >= 'a') && (c <= 'f')) ? (uint8_t)(c - 'a' + 10u) : \
((c >= 'A') && (c <= 'F')) ? (uint8_t)(c - 'A' + 10u) : 0u)

Convert hexadecimal digit to integer.
