21/*
22 * Security protocols EAPOL key storage module. This is used by EAPOL protocols to store
23 * and update key information. This can be used either as supplicant key storage or as
24 * an authenticator key storage for a specific supplicant. Storage can be also used to
25 * access global security data (Group Transient Keys and certificate information).
26 *
27 */
