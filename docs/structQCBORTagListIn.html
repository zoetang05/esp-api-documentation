QCBORTagListIn Struct Reference

#include <qcbor.h>

Public Attributes

uint8_t uNumTags
 
const uint64_t * puTags
 

Detailed Description

This is used to tell the decoder about tags that it should record in uTagBits in QCBORItem beyond the built-in tags. puTags points to an array of uint64_t integers that are the tags. uNumTags is the number of integers in the array. The maximum size is QCBOR_MAX_CUSTOM_TAGS. See QCBORDecode_IsTagged() and QCBORDecode_SetCallerAddedTagMap().
