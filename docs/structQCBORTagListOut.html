Detailed Description

This is for QCBORDecode_GetNextWithTags() to be able to return the full list of tags on an item. It not needed for most CBOR protocol implementations. Its primary use is for pretty-printing CBOR or protocol conversion to another format.

On input, puTags points to a buffer to be filled in and uNumAllocated is the number of uint64_t values in the buffer.

On output the buffer contains the tags for the item. uNumUsed tells how many there are.
