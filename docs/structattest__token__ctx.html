Detailed Description

The context for creating an attestation token. The caller of attest_token must create one of these and pass it to the functions here. It is small enough that it can go on the stack. It is most of the memory needed to create a token except the output buffer and any memory requirements for the cryptographic operations.

The structure is opaque for the caller.

This is roughly 148 + 8 + 32 = 188 bytes
