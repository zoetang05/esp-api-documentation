Detailed Description

The Write Response is sent in reply to a valid Write Request and acknowledges that the attribute has been successfully written. It is just a placeholder which indicates the client that the write request was successful.

Note
see BLUETOOTH SPECIFICATION Version 5.0 | Vol 3, Part F Section 3.4.5.2
