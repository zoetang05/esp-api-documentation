◆ use_legacy_pdu

bool ble::advertising_event_properties_t::use_legacy_pdu

If set advertising packets use legacy advertising PDU format and the members omit_advertisser_address and include_tx_power are ignored. If not set then:

  • The advertisement can't be both connectable and scannable.
  • High duty cycle directed connectable advertising can't be use.
