enum ble::privacy_mode_t::type
Enumerator
NETWORK 

In Network mode, an observer or peripheral does not accept advertising packets containing the identity address of peer devices that have distributed their IRK

DEVICE 

In this mode, the device is only concerned about its own privacy. This mode should only be used when the Resolvable Private Address Only characteristic is not present in the GAP service of the peer device.
