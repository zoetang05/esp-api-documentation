◆ chunk_size

uint8_t radio_fsk_packet_handler::chunk_size

Defines the size of a chunk of outgoing buffer written to the FIFO at a unit time. For example, if the size of the data exceeds the FIFO limit, a certain sized chunk is written to the FIFO. Later, a FIFO-level interrupt enables writing of the remaining data to the FIFO chunk by chunk until transmission is complete.
