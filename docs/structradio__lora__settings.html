bool radio_lora_settings::iq_inverted

Turn IQ inversion on/off. Usually, the end device sends an IQ inverted signal, and the base stations do not invert. We recommended sending an IQ inverted signal from the device side, so any transmissions from the base stations do not interfere with end device transmission.
