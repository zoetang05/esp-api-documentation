Detailed Description

This file serves as the wrapper for the platform/OS dependent functions It is needed to modify these functions accordingly based on the platform and the OS. Whenever the synopsys GMAC driver ported on to different platform, this file should be handled at most care. The corresponding function definitions for non-inline functions are available in synopGMAC_plat.c file.
