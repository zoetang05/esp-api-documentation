This the payload of TL_Evt_t for a command status event

This the payload of TL_Evt_t for a command complete event

This the payload of TL_Evt_t for an asynchronous event

This format shall be used for all events (asynchronous and command response) reported by the CPU2 except for the command response of a system command where the header is not there and the format to be used shall be TL_EvtSerial_t. Note: Be careful that the asynchronous events reported by the CPU2 on the system channel do include the header and shall use TL_EvtPacket_t format. Only the command response format on the system channel is different.
