◆ whiteboard_set_device_hard_limit()

void whiteboard_set_device_hard_limit ( uint16_t limit)
extern

Whiteboard_set_device_hard_limit Sets the absolut limit of child devices this device can handle. This is very useful in situations, where 1 GW drops from network and causes it's children to join other GW networks. This might cause other GWs to run out of memory.

Parameters
limitAbsolute maximum amount of devices allowed to join. Default value=0 means unlimited (as long as there is memory)
