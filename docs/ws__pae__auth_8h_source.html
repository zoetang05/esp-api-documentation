23/*
24 * Authenticator port access entity controls key security protocols using KMP API.
25 *
26 * Configures KMP service network access and provides timing and callback services
27 * for it. Registers needed security protocols to KMP service.
28 *
29 * PAE Maintains security keys that are internal to port access entity for
30 * each supplicant and maintains supplicant security registration state.
31 *
32 * Autenticator PAE controls network access keys and provides new network
33 * access keys to supplicants when they are changed. When supplicant
34 * network keys are updated, also other keys (master key, pairwise key)
35 * are updated as needed.
36 *
37 */
